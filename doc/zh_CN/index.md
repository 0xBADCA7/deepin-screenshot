# 深度截图|deepin-screenshot.svg|
## 概述|overview.svg|
深度截图是武汉深之度科技有限公司开发的深度操作系统下自带的截图工具。在日常工作和娱乐中，为了展示特定状态下的程序界面，应用场景等，需要截取可以显示在屏幕或其他显示设备上的可视图像，您可以通过运行深度截图软件来实现。

深度截图拥有模糊和马赛克的图片编辑功能，在传播图片的过程中帮助您保护隐私信息不外泄，支持丰富的网络社交平台，让您随时随地的与朋友互动，分享身边的精彩！

 ![1|深度截图](deepin-shot.png)

您可以通过多种方式来实现快速截图：

* 使用键盘上的 Print 键来截取整个显示器的屏幕。

* 使用各个应用软件如腾讯 QQ、chrome 浏览器等软件的截图功能。

* 使用专业的截图软件来截取屏幕，如深度操作系统自带的深度截图软件。

## 基本操作|basic-operation.svg|
### 运行深度截图
您可以通过执行以下操作之一来运行深度截图，进入截图模式后，整个电脑屏幕将会变暗。

* 如果您已经将深度截图固定到桌面，在桌面，点击 ![深度截图](deepin-screenshot25px.png)。

* 如果您已经将深度截图固定到任务栏，在任务栏，点击 ![深度截图](deepin-screenshot25px.png)。

* 在任务栏上，点击 ![启动器](start-here25px.png) 进入启动器界面，通过浏览找到 ![深度截图](deepin-screenshot25px.png) 点击。

* 按下键盘上的 Ctrl+Alt+A 组合键来快速运行深度截图。


### 延时截图

延时截图给您预留了5s的准备时间，帮您截取最精准的瞬间。

> **注意：** 如果您已经将深度截图固定到任务栏，右键单击任务栏上的 ![深度截图](deepin-screenshot25px.png)，然后选择 **延时5s** 来延时截取。

您也可以通过操作以下步骤来实现延时截图：

1. 按下键盘上的 Ctrl+Print 组合键。

2. 在屏幕的右上角，您会看到“深度截图将在5秒后启动”的提示信息。

3. 等待5s后自动进入截图模式，整个屏幕变暗。

### 退出深度截图

在截图模式下，您可以通过以下两种方式来退出截图：

* 按下键盘上的 Esc 键。

* 点击鼠标右键，退出截图模式。

## 选择截图区域|select-screenshot-area.svg|

目前比较常用的有三种截图方式：全屏截图、程序窗口截图和自选区域截图。当选中截取范围后，在截图区域的四周会出现一个蓝色边框，并且该区域会恢复亮度显示。

> **注意：** 当电脑多屏显示时，您也可以使用深度截图来截取不同屏幕上的信息。

### 全屏截图

全屏截图是自动截取当前显示器的整个屏幕。

![1|全屏截图](fullscreen.png)

* 如果您已经将深度截图固定到任务栏，右键单击任务栏上的 ![深度截图](deepin-screenshot25px.png)，然后选择 **全屏截图** 来截取全屏。

* 按下键盘上的 Print 键实现全屏截图。

您也可以通过操作以下步骤来实现全屏截图：

1. 按下键盘上的 Ctrl+Alt+A 组合键，进入截图模式。

2. 将鼠标指针移至桌面上，深度截图会自动选中整个屏幕，并在其左上角显示当前截图区域的尺寸大小。

3. 点击桌面，在桌面的右上角会弹出工具栏。

4. 如果要退出截图，请点击工具栏上的 ![关闭](cancel25px.png) 或点击鼠标右键，选择 **退出**。


### 程序窗口截图
程序窗口截图是自动识别当前的应用窗口来截取整个窗口。

![1|窗口截图](window.png)

1. 按下键盘上的 Ctrl+Alt+A 组合键，进入截图模式。

2. 将鼠标指针移至打开的应用窗口上，深度截图会自动选中该窗口，并在其左上角显示当前截图区域的尺寸大小。

3. 点击窗口，在窗口的右下角会弹出工具栏。

4. 如果要退出截图，请点击工具栏上的 ![关闭](cancel25px.png) 或点击鼠标右键，选择 **退出**。

### 自选区域截图
自选区域截图是通过拖动鼠标，自由选择截取范围的截图方式。

![1|区域截图](part-area.png)

1. 按下键盘上的 Ctrl+Alt+A 组合键，进入截图模式。

2. 按住鼠标左键不放，拖动鼠标选择截图区域，在其左上角将实时显示当前截图区域的尺寸大小。

3. 释放鼠标左键，完成截图，在截图区域的右下角会弹出工具栏。

4. 如果要退出截图，请点击工具栏上的 ![关闭](cancel25px.png) 或点击鼠标右键，选择 **退出**。

## 调整截图区域|adjust-screenshot.svg|

您可以对截图区域进行微调，例如放大或缩小截取范围，移动截图等。

### 放大/ 缩小截图区域
将鼠标指针置于截图区域的蓝色边框上，鼠标指针变为 ![双箭头](double-arrow25px.png)，边框会变成点状边缘。您可以：

* 按住鼠标左键不放，拖动鼠标来放大或缩小截图区域。

* 按下键盘上的 Ctrl+![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下扩展截图区域，按下键盘上的 Ctrl+![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右扩展截图区域。

### 移动截图位置
将鼠标指针置于截图区域上，鼠标指针变为 ![手指](finger25px.png)。您可以：

* 按住鼠标左键不放，拖动鼠标来移动截图区域的位置。

* 按下键盘上的 ![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下移动截图区域，按下键盘上的 ![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右移动截图区域。

### 绘制图形
您可以在截取的图片中绘制一些简单的图形，如矩形、椭圆等。

#### 矩形工具

![1|矩形工具](rect-tool.png)

1. 在截图区域下方的工具栏中，点击 ![矩形](rect25px.png)。

2. 在工具栏展开面板中，选择矩形边线的粗细。

3. 将鼠标指针置于截图区域上，鼠标指针变为 ![矩形指针](rect_mouse25px.png)。

4. 按住鼠标左键不放，拖动鼠标以完成图形区域的选择。

5. 如果截图中包含了个人隐私信息，请点击工具栏展开面板中的 ![模糊](blur25px.png) 或 ![马赛克](mosaic25px.png) 来涂抹。

![2|模糊工具](blur-tool.png)
![2|马赛克工具](mosaic-tool.png)

#### 椭圆工具

![1|椭圆工具](ellipse-tool.png)

1. 在截图区域下方的工具栏中，点击 ![椭圆](ellipse25px.png)。

2. 在工具栏展开面板中，选择椭圆边线的粗细。

3. 将鼠标指针置于截图区域上，鼠标指针变为 ![椭圆指针](ellipse_mouse25px.png)。

4. 按住鼠标左键不放，拖动鼠标以完成图形区域的选择。

5. 如果截图中包含了个人隐私信息，请点击工具栏展开面板中的 ![模糊](blur25px.png) 或 ![马赛克](mosaic25px.png) 来涂抹。

#### 箭头工具

![1|箭头工具](arrow-tool.png)

1. 在截图区域下方的工具栏中，点击 ![箭头](arrow25px.png)。

2. 在工具栏展开面板中，选择箭头的粗细。

3. 将鼠标指针置于截图区域上，鼠标指针变为 ![箭头指针](arrow_mouse25px.png)。

4. 按住鼠标左键不放，拖动鼠标以完成图形区域的选择。

#### 画笔工具

![1|画笔工具](line-tool.png)

1. 在截图区域下方的工具栏中，点击 ![画笔](line25px.png)。

2. 在工具栏展开面板中，选择画笔的粗细。

3. 点击 ![画笔选择](straightline25px.png)，在自由画笔和直线画笔之间切换。

4. 将鼠标指针置于截图区域上，鼠标指针变为 ![画笔指针](pink_red25px.png)。

5. 按住鼠标左键不放，拖动鼠标以完成图形区域的选择。

> **注意：** 如果需要绘制正方形、圆形等图形，您可以：
  * 按住键盘上的 Shift 键，然后选择 ![矩形](rect25px.png)，绘制正方形。
  * 按住键盘上的 Shift 键，然后选择 ![椭圆](ellipse25px.png)，绘制圆形。
  * 按住键盘上的 Shift 键，然后选择 ![箭头](arrow25px.png)，绘制水平或垂直方向的箭头。
  * 按住键盘上的 Shift 键，然后选择 ![画笔](line25px.png)，绘制水平或垂直方向的直线。

### 修改图形
如果您在截图中绘制了其他的图形，您可以对绘制的图形进行修改和移动。

![1|修改图形](change-tool.png)

#### 移动图形
1. 将鼠标指针置于图形的边线上，此时鼠标指针变为 ![手指](finger25px.png)。

2. 按住鼠标左键不放，可以将图形移动到截图区域的任何位置。

3. 按下键盘上的 Ctrl+![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下扩展图形区域，按下键盘上的 Ctrl+![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右扩展图形区域。

4. 按下键盘上的 ![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下移动图形，按下键盘上的 ![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右移动图形。

#### 编辑图形
1. 将鼠标指针置于图形的边线上。

2. 点击鼠标左键，进入图形的编辑模式，您可以：

   - 按下键盘上的 Delete 键，删除图形。
   
   - 将鼠标指针置于编辑框上，此时鼠标指针变成 ![双箭头](double-arrow25px.png)，通过拖动鼠标指针，来放大或缩小图形区域。
   
   - 将鼠标指针置于编辑框的 ![旋转](rotate25px.png) 上，此时鼠标指针变成 ![旋转](rotate_mouse25px.png)，通过拖动鼠标指针，来旋转图形。
   
3. 在编辑框外点击鼠标左键，退出编辑模式。

### 添加文字批注
对截取的图片进行文字补充和说明，帮助他人更清楚的了解截取的图片。

![1|文本框](text-tool.png)

1. 在截图区域下方的工具栏中，点击 ![文字](text25px.png)。

2. 在工具栏展开面板中，输入字号大小或通过点击 ![增大](add25px.png) 或 ![减小](minus25px.png) 来增大或减小批注的字体大小。

3. 将鼠标指针置于截取的图片上，此时鼠标指针变为 ![文字指针](text_mouse25px.png)。

4. 点击要添加批注的地方，将出现一个待输入的文本框。

5. 在文本框中输入文字。

### 修改文字批注
如果您已经添加了文字批注，您可以对文字批注进行修改和移动。

#### 移动文字批注
1. 将鼠标指针置于文字上，此时鼠标指针变为 ![手指](finger25px.png)。

2. 按住鼠标左键不放，可以将文字移动到截图区域的任何位置。

3. 按下键盘上的 Ctrl+![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下扩展文本框，按下键盘上的 Ctrl+![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右扩展文本框。

4. 按下键盘上的 ![向上](Up25px.png) 或 ![向下](Down25px.png) 来上下移动文本框，按下键盘上的 ![向左](Left25px.png) 或 ![向右](Right25px.png) 来左右移动文本框。

#### 编辑文字批注
1. 将鼠标指针置于文字上。

2. 点击鼠标左键，进入文字编辑模式，您可以：

   - 按下键盘上的 Delete 键，删除文字批注。
   
   - 在文本框内，修改文字信息。
   
   - 将鼠标指针置于文本框的 ![旋转](rotate25px.png) 上，此时鼠标指针变成 ![旋转](rotate_mouse25px.png)，通过拖动鼠标指针，来旋转文字批注。
   
3. 在文本框外点击鼠标左键，退出编辑模式。

## 保存截图|save-screenshot.svg|
将截取的图片保存下来，为后续的使用储存素材！

当截图保存成功之后，您可以在桌面右上角看到提示信息，点击 **查看**，可打开截图所在的文件夹。

您可以通过执行以下操作之一来保存截取的图片：

* 按下键盘上的 Ctrl+S 组合键来保存。

* 按下键盘上的 Enter 键来直接保存。

* 在截取的图片中，点击鼠标右键，选择 **保存** 或双击鼠标完成保存操作。

> **注意：** 在以上操作中，截取的图片默认存放到系统的图片文件夹下。

您还可以在截图区域下方的工具栏中，点击 ![保存](save25px.png) 的下拉选项：

![0|保存](save-tool.png)

## 分享截图|share-screenshot.svg|
您可以将有趣的图片截取下来，配上文字，将这份快乐传递给您的好友，让更多人感受到！

您可以按下键盘上的 Ctrl+Enter 组合键，快速实现分享！您也可以执行以下操作来分享：

1. 在截图区域下方的工具栏中，点击 ![分享](share25px.png)。

2. 在分享界面中，输入您想说的文字，点击 **下一步**。

3. 选择一个社交账号，点击 **分享**。

4. 登录您的社交账号授权并分享。

5. 当分享成功之后，您可以在桌面右上角看到提示信息。

6. 如果分享失败，请点击 **重试** 再次分享。

7. 如果要退出分享，请点击 **取消** 或 **关闭**。

> **注意：** 在分享界面输入的文字请控制在140字以内。

![1|分享](share.png)

### 管理多个账号
当分享截图后，深度截图会记住您的账号与密码，以便下次的分享操作，您可以对多个账号进行管理。

在分享界面中，点击左下角的 **账户管理**，您可以：

* 注销登录账号。

* 使用一个新的账号来登录分享。
